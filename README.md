# VIT Semester Result — Spring Boot (MongoDB) + React

This implementation adapts the assignment (pages 11–end) to use **MongoDB** instead of MySQL.
- Backend: Spring Boot 3.3, Spring Data MongoDB
- Frontend: React + Vite (Bootstrap via CDN)

## How to run
1) Start MongoDB locally (default: `mongodb://localhost:27017`).
2) Backend:
```bash
cd backend
mvn spring-boot:run
```
3) Frontend:
```bash
cd frontend
npm install
npm run dev
```

Then open http://localhost:5173 and submit the form.
